Instagram URL Extractor

Extract post URLs and image URLs from any Instagram profile

Method 1: Browser Console EASIEST

Open Instagram profile: https://www.instagram.com/heartlands.studio/
Scroll down to load about 40 photos
Press F12 to open Developer Tools
Click the Console tab
Paste the code below and press Enter
// Extract Post URLs const postLinks = Array.from(document.querySelectorAll('a[href*="/p/"]')) .map(a => a.href) .filter(href => href.includes('/p/')) .filter((v, i, a) => a.indexOf(v) === i); // Extract Image URLs const imageUrls = Array.from(document.querySelectorAll('img[src*="cdninstagram"]')) .map(img => img.src) .filter(src => src.includes('cdninstagram')) .filter((v, i, a) => a.indexOf(v) === i); console.log('=== POST URLS ==='); console.log(postLinks.join('\n')); console.log('\n=== IMAGE URLS ==='); console.log(imageUrls.join('\n')); // Copy to clipboard const output = '// Post URLs:\n' + postLinks.join('\n') + '\n\n// Image URLs:\n' + imageUrls.join('\n'); copy(output); alert(`‚úì Extracted ${postLinks.length} posts and ${imageUrls.length} images!\nURLs copied to clipboard!`);
‚úì This will automatically copy all URLs to your clipboard!

Method 2: Generate JavaScript Array DIRECT

Use Method 1 to get image URLs
Or paste the code below in the console for a ready-to-use array:
// Extract and format as JavaScript array const imageUrls = Array.from(document.querySelectorAll('img[src*="cdninstagram"]')) .map(img => img.src) .filter(src => src.includes('cdninstagram')) .filter((v, i, a) => a.indexOf(v) === i); const jsArray = 'const ACCOUNT_A_PHOTOS = [\n' + imageUrls.map(url => ` '${url}',`).join('\n') + '\n];'; console.log(jsArray); copy(jsArray); alert(`‚úì Generated JavaScript array with ${imageUrls.length} images!\nCode copied - paste directly into script.js!`);
‚úì This creates a ready-to-paste JavaScript array for your web app!
